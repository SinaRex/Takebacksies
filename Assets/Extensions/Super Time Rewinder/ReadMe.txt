This package provides a scripting system for manage time rewind.<BR>
Now it's very easy and quick to implement in your game a time rewind system. You only need to attach one script on all objects that could rewind in the time. <BR>
You can set maximum rewind time and keep or not the velocity of the given object after rewinding's exit. 
BR><BR>

<STRONG>Package includes:</STRONG><BR>

- 2 scripts, shoold be apply on objects (1 for 3D and 1 for 2D) .<BR> 
- 3 DEMO scenes.<BR>
</BR></BR>
